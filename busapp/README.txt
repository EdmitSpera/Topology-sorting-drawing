1. 概述
欢迎使用图形可视化应用程序！这个应用程序旨在帮助您可视化和分析有向图和带权有向图的数据。您可以使用这个应用程序来执行以下操作：

导入图数据
生成无权有向图
生成带权有向图
执行拓扑排序
计算最短路径

3. 主要功能
3.1 导入图数据
您可以通过菜单中的 "File" -> "Open" 选项来导入图数据。选择一个本地文本文件，其格式应符合应用程序的要求，以便正确导入数据。

3.2 生成无权有向图
在应用程序中，您可以输入关系数据，然后点击 "生成无权图" 按钮来生成无权有向图。关系数据的格式应为 <a,b> 或 《a,b》，其中 a 和 b 是顶点的名称。

3.3 生成带权有向图
与无权图类似，您可以输入带权边的数据，然后点击 "生成带权图" 按钮来生成带权有向图。带权边的数据格式应为 <a,b,1> 或 《a,b,1》，其中 a 和 b 是顶点的名称，1 是边的权重。

3.4 执行拓扑排序
一旦生成了无权有向图，您可以点击 "生成图" 按钮来执行拓扑排序，并可视化显示排序结果。排序结果将显示在应用程序的图形界面上，并且可以在文本框中看到。

3.5 计算最短路径
如果您已经生成了带权有向图，您可以点击 "最短路径" 按钮来计算最短路径。您需要输入起始点的名称，然后应用程序将计算从该起始点出发到其他所有点的最短路径，并在文本框中显示结果。

3.6 保存图像和结果
您可以使用菜单中的 "File" -> "Save Graph" 选项来保存生成的图像。同样地，您也可以使用 "File" -> "Export Results" 选项来将排序结果导出为文本文件。

4. 使用方法
以下是一些基本的使用方法：

打开应用程序后，使用菜单中的 "File" -> "Open" 选项导入图数据。

输入关系数据或带权边数据，然后点击 "生成无权图" 或 "生成带权图" 按钮。

如果您生成了无权图，请点击 "生成图" 按钮执行拓扑排序。

如果您生成了带权图，可以计算最短路径，首先输入起始点的名称，然后点击 "最短路径" 按钮。

您可以使用菜单中的 "File" -> "Save Graph" 选项保存图像，或使用 "File" -> "Export Results" 选项将排序结果导出为文本文件。

5. 常见问题解答
5.1. 数据导入问题
如果无法导入数据，请确保您的数据文件格式正确，并且数据与应用程序的要求一致。

5.2. 图生成问题
如果无法生成图，请检查输入的关系数据或带权边数据的格式是否正确。应用程序要求数据的格式为 <a,b> 或 《a,b》（无权图）或 <a,b,1> 或 《a,b,1》（带权图）。

5.3. 最短路径计算问题
如果无法计算最短路径，请确保已经生成了带权图，并输入了正确的起始点名称。
